# A propos

Cet outil aide à convertir les données venant d'une base de donnée pronote afin de créer des classes wims. C'est en quelque sorte un convertisseur de fichier csv de pronote vers wims.

Aucune donnée n'est stoquée ou transmise et le code est disponible sur ce dépot github sous licence GPL-3.

# Un problème? Une question?

N'hésitez pas à créer une issue dans le dépot github de ce projet.

# Utilisation

# Exporter les classes à partir de pronote

Dans pronote, allez dans un onglet affichant une liste d'élèves comme notes ou classes. Cliquez sur l'icone et enregistrez le fichier csv.

# Importer dans pronote2wims

Dans données pronote cliquer sur puis séléctionner le fichier CSV téléchargé précedement. Envoyer le fichier en cliquant sur le bouton

# Configurer

Vous pouvez choisir le format de l'identifiant et du mot de passe qui sera généré pour une utilisation dans wims.

Mot de passe
  • Prénom: Le mot de passe sera le prénom de l'utilisateur. Il commence par une majuscule et ne contient pas d'accents.
  • Fixe: Le mot de passe sera le même pour tous les utilisateurs. Vous pouvez choisir le mot de passe dans le champ Même pour tous.
  • Aléatoire: Un mot de passe alphanumérique aléatoire sera généré pour chaque utilisateur. Vous pouvez choisir ta taille du mot de passe.
Identifiant

Vous pouvez choisir le format de l'identifiant.

# Prévisualiser

Après avoir modifié la configuration, cliquez sur pour voir les données dans le tableau de droite.

# Corriger les erreurs

Vous pouvez corriger les erreurs directement en cliquant dessus (au survol, la raison de l'erreur est expliquée).

Une fois que toutes les erreurs ont disparu, un fichier contenant les identifiants vous est proposé au téléchargement.

# Importer les identifiants dans Wims

Il ne vous reste plus qu'à vous rendre dans votre classe Wims, rubrique "Gestion" > "liaison tableur".

Wims vous propose d'envoyer vos identifiants dans la classe. Pensez à vérifier la liste des identifiants reconnus pour vérifier qu'il n'y a aucune erreur à ce stade...
